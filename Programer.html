Projet SAÉ 1.04 : Création d'un site web portfolio (2024)

Objectif principal : Développement d'un site web personnel présentant mon curriculum vitæ avec :

  • Une page principale structurée sémantiquement (HTML5)
  • Minimum deux pages secondaires liées
  • Une feuille de style CSS unique pour l'ensemble du site

Structure mise en œuvre :

  • Header : Informations personnelles avec photo ronde, résumé professionnel (2-3 lignes), liens interactifs (Google Maps, mailto)
  • Section Formations : Résumé synthétique avec ancres vers page détaillée (spécialités, modules, projets réalisés)
  • Section Diplômes : Présentation des diplômes avec liens vers établissements et résultats détaillés
  • Section Expériences : Résumé des expériences professionnelles avec liens vers détails (postes, responsabilités)
  • Section Compétences :
    • Flexbox à trois colonnes (compétences RT, techniques, humaines)
    • Section complémentaire (langues, sports, hobbies, activités culturelles)

Techniques maîtrisées :

  • Navigation par ancres (#nom_ancre) au sein d'une même page
  • Liens interpages avec ancres ciblées (page.html#section)
  • Mise en œuvre de scroll-behavior: smooth pour défilement fluide
  • Intégration de liens interactifs (mailto, Google Maps)
  • Gestion de projet sous Codepen.io avec export local

Contraintes techniques respectées :

  • Validation W3C du code HTML/CSS
  • Design responsive (adaptatif)
  • Optimisation pour référencement (balises sémantiques)
  • Gestion des chemins relatifs pour l'export local

Apprentissages clés : Ce projet m'a permis de consolider mes compétences en développement front-end, particulièrement dans la création d'interfaces structurées et accessibles. J'ai appris à organiser un projet multi-pages avec une architecture cohérente et une navigation intuitive, tout en respectant les bonnes pratiques du web sémantique.

Validation de la compétence "Programmer" : Ce projet a démontré ma maîtrise des fondamentaux de la programmation web front-end. J'ai implémenté :

  • Une structure HTML5 sémantique rigoureuse avec balises <header>, <section>
  • Des algorithmes de navigation complexes via système d'ancres intra/inter-pages
  • Un design responsive utilisant Flexbox pour la mise en page des compétences
  • Des fonctions JavaScript pour les interactions (mailto, Google Maps)
  • Une feuille de style CSS unique gérant l'ensemble du site

J'ai optimisé le code selon les standards W3C et développé des solutions techniques comme le scroll-behavior smooth pour améliorer l'UX.

Projet SAÉ 1.05 : Développement d'un outil de reporting pour l'analyse de disque (2024)

Objectif principal : Création d'un outil permettant d'identifier et localiser les fichiers volumineux sur un disque dur pour optimiser l'espace de stockage.

Fonctionnalités clés développées :

  • Analyse récursive de l'arborescence des répertoires
  • Identification des 100 fichiers les plus volumineux (> seuil personnalisable)
  • Génération d'un rapport JSON structuré
  • Interface graphique avec visualisation en camembert
  • Système de légendes paginées avec cases à cocher
  • Génération automatique de scripts PowerShell de suppression

Technologies maîtrisées :

  • Python (traitement de fichiers, tri de données)
  • PyQt5 (interface graphique)
  • JSON (stockage et échange de données)
  • PowerShell (automatisation système)
  • Bibliothèque pathlib (gestion multi-plateforme des chemins)

Composants techniques :

  • Script PowerShell : Orchestration du processus
  • Analyseur Python :
    • Parcours récursif des répertoires
    • Tri décroissant par taille de fichier
    • Filtrage par seuil (1Mo/10Mo/etc.)
    • Export des résultats en JSON
  • Interface graphique :
    • Camembert interactif visualisant la répartition des fichiers
    • Légendes paginées (25 par page)
    • Système de cases à cocher pour sélection des fichiers
    • Génération de scripts de suppression

Défis techniques relevés :

  • Gestion des spécificités Windows (antislashes dans les chemins)
  • Optimisation des performances pour les grandes arborescences
  • Création d'une interface responsive avec onglets
  • Génération dynamique de scripts PowerShell multi-plateforme
  • Implémentation de mécanismes de sécurité (confirmations de suppression)
Présentation final de la SAE sous format pdf

Apprentissages clés : Ce projet m'a permis de développer des compétences avancées en traitement de données et en création d'outils système. J'ai notamment consolidé mes connaissances en programmation Python orientée objet, en développement d'interfaces graphiques complexes avec PyQt5, et en automatisation de tâches d'administration système via PowerShell.

Validation de la compétence "Programmer" : Ce projet a exigé des compétences avancées en programmation Python :

  • Algorithmes de parcours récursif d'arborescences avec pathlib
  • Tri et filtrage de datasets complexes via fonctions lambda
  • Manipulation précise de JSON (sérialisation/désérialisation)
  • Programmation orientée objet avec PyQt5 (classes Camembert, Legendes)
  • Gestion d'événements et callbacks pour l'interface graphique
  • Génération dynamique de scripts PowerShell

J'ai résolu des défis algorithmiques comme le pagination des légendes et l'optimisation des tris sur grands datasets.
