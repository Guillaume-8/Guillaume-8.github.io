SAE 104 : Création d'un site web portfolio

Python 3 HTML3/CSS JavaScript

Description du projet

L'objectif de cette SAE était de concevoir un portfolio, pour cela nous avons eu le choix de le faire de nos propre moyens en le codant en html/css ou alors de le faire avec le avec le site web Mahara. J'ai choisi de le faire de mes propre mains et celui que vous liser actuelement en est la deuxieme version.

Objectifs Techniques

  • Navigation pas ancre : sous la forme "#nom-ancre", cela sert au seuin d'une même page de ce déplacer rapidement. Dans le cadre de ma page web, les ancres était positionnées en bas de page qui permetaient de revenir en haut de page une fois la page lu.
  • Liens interpages : On avais pour contrainte de n'utiliser qu'une seule page CSS pour l'ensemble du site et il fallais un minimum de 2 pages secondaires. J'en ai donc fait plusieurs (au nombre de 5) avec une seule page CSS.
  • Intégration de liens : Sacahant que c'est un portfolio et qu'il contient des projet j'ai donc dû mettre des liens avec des preuves de réusite ou de problème pour attester du bon déroulé du projet.

Apprentissages clés

Ce projet m'a permis de consolider mes compétences en développement front-end, particulièrement dans la création d'interfaces structurées et accessibles. J'ai appris à organiser un projet multi-pages avec une architecture cohérente et une navigation intuitive, tout en respectant les bonnes pratiques du web sémantique.

Lien avec la Cybersécurité

Apprendre à programmer proprement est indispensable pour la Cyber :

  • Sécurisation des entrées : Éviter les plantages ou les injections via des contrôles stricts des données.

Validation de la compétence "Programmer"

Ce projet a démontré ma maîtrise des fondamentaux de la programmation web front-end. J'ai implémenté :


  • Une structure HTML5 sémantique rigoureuse avec balises <header>, <section>
  • Des algorithmes de navigation complexes via système d'ancres intra/inter-pages
  • Un design responsive utilisant Flexbox pour la mise en page des compétences
  • Des fonctions JavaScript pour les interactions (mailto, Google Maps)
  • Une feuille de style CSS unique gérant l'ensemble du site
