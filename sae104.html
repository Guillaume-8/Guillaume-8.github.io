SAE 104 : Création d'un site web portfolio

Python 3 HTML3/CSS JavaScript

Description du projet

L'objectif de cette SAE était de concevoir un portfolio. J'ai choisi de le coder entièrement à la main en HTML/CSS, et la version que vous lisez actuellement en est la deuxième itération, optimisée et stylisée.

Objectifs Techniques

  • Navigation par ancre : Utilisation de liens internes pour faciliter la navigation fluide au sein des pages.
  • Architecture Multi-pages : Gestion de 5 pages distinctes reliées à une feuille de style CSS unique et cohérente.
  • Intégration de ressources : Mise en place de liens de téléchargement et de preuves de réalisation.
