SAE 301 : Prise Connectée Intelligente Wifi

ESP8266 MQTT (Mosquitto) Raspberry Pi Android / Web Électronique (EasyEDA)

Architecture & Design

Ce projet consiste à concevoir une prise électrique intelligente pilotable à distance. L'architecture repose sur un écosystème IoT complet : une carte électronique sur mesure, un serveur Web sur Raspberry Pi et une application mobile pour le contrôle utilisateur.

Matériel (Hardware)

Conception du PCB sur EasyEDA intégrant un module ESP8266, un capteur de température DS18B20 et un relais pour la commande de puissance.

Communication IoT

Utilisation du protocole MQTT via un broker (Mosquitto) pour assurer la liaison en temps réel entre la prise et les interfaces de commande.

Flux de données MQTT

Aperçu des messages échangés sur le broker pour piloter la prise et remonter la température :

# Souscription aux états de la prise
$ mosquitto_sub -h localhost -t "maison/prise1/etat"
ON

# Lecture de la température envoyée par le DS18B20
$ mosquitto_sub -h localhost -t "maison/prise1/temp"
22.5°C
Dimension Cybersécurité : Le projet intègre la sécurisation des accès aux interfaces Web et Android. J'ai mis en œuvre la sécurisation du transfert de données côté Internet et l'utilisation d'un broker privé pour isoler les flux de commande.

Compétences Validées

  • Développement Fullstack : Création d'un serveur Web sur Raspberry Pi et d'une application Android avec gestion de plages horaires.
  • Ingénierie Système : Configuration de l'ESP8266 en mode AP (Access Point) pour la configuration initiale du Wifi.
  • Notification & Alerting : Mise en place d'alertes SMS/Mail via Pushbullet/Gammu en cas de dépassement de seuil de température.
