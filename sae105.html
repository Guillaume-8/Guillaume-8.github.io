SAE 105 : Développement d'un outil de reporting pour l'analyse de disque

PowerShell Analyse de données Fichiers CSV/JSON Python

Objectifs du projet

Création d'un outil permettant d'identifier et localiser les fichiers volumineux sur un disque dur pour optimiser l'espace de stockage.

Fonctionnalités clés développées

    Le projet intègre une analyse récursive complète de l'arborescence des répertoires, permettant d'explorer chaque sous-dossier en profondeur. Cette exploration aboutit à l'identification précise des 100 fichiers les plus volumineux, basée sur un seuil de taille entièrement personnalisable par l'utilisateur. Pour faciliter l'exploitation des données, l'outil génère un rapport JSON structuré et propose une interface graphique intuitive incluant une visualisation en camembert. La navigation est optimisée par un système de légendes paginées avec cases à cocher, complété par une fonctionnalité de génération automatique de scripts PowerShell dédiés à la suppression sécurisée des fichiers sélectionnés.

Technologies maîtrisées

Le développement de cette solution repose sur un socle technique solide :

  • Python : utilisé comme langage principal pour le traitement complexe des fichiers et le tri performant des données.
  • PyQt5 : pour la conception d'une interface graphique moderne et interactive.
  • JSON : employé pour assurer un stockage et un échange de données structuré et léger.
  • PowerShell : intégré pour l'automatisation système et l'exécution des tâches d'administration.
  • Bibliothèque pathlib : mise en œuvre pour garantir une gestion multi-plateforme rigoureuse des chemins de fichiers.

Composants techniques

L'architecture de l'application s'articule autour de trois piliers majeurs:

  • Script PowerShell :Il assure l'orchestration globale du processus de l'application.
  • Analyseur Python : Ce moteur effectue le parcours récursif, le tri décroissant par taille de fichier, ainsi que le filtrage selon les seuils définis (1Mo, 10Mo, etc.) avant d'exporter les résultats au format JSON.
  • Interface graphique : Elle offre un camembert interactif pour visualiser la répartition des fichiers, des légendes paginées (25 éléments par page), et un système de cases à cocher permettant la sélection précise des fichiers en vue de la génération de scripts de suppression.

Apprentissages clés

Ce projet m'a permis de développer des compétences avancées en traitement de données et en création d'outils système. J'ai notamment consolidé mes connaissances en programmation Python orientée objet, en développement d'interfaces graphiques complexes avec PyQt5, et en automatisation de tâches d'administration système via PowerShell.

Validation de la compétence "Programmer"

Ce projet a exigé des compétences avancées en programmation Python :

  • Algorithmes de parcours récursif d'arborescences avec pathlib
  • Tri et filtrage de datasets complexes via fonctions lambda
  • Manipulation précise de JSON (sérialisation/désérialisation)
  • Programmation orientée objet avec PyQt5 (classes Camembert, Legendes)
  • Gestion d'événements et callbacks pour l'interface graphique
  • Génération dynamique de scripts PowerShell
